- 🔄 **Pembuatan akun massal** – tentukan jumlah akun yang diinginkan
- 🎨 **UI Premium** – panel, tree, spinner, warna via **Rich**
- 👤 **Data acak realistis** – nama, tanggal lahir, gender, foto profil & cover (Faker)
- 🌐 **Multi-bahasa** – Indonesia, Inggris, Mandarin, custom
- 🔐 **Password manual / otomatis** – sesuai pilihan
- 📧 **Verifikasi email manual** – kode OTP dimasukkan langsung
- 💾 **Hasil tersimpan otomatis** – file `AcounttFresh.txt` di `/sdcard/Premium/CreateFbrif`

## 📦 Instalasi Lengkap per Platform

### 📱 Termux (Android)
```bash

🚀 Cara Menjalankan (umum)

1. Buka terminal di folder CreateFacebookv2.
2. Jalankan:
   ```bash
   python CreateFacebook2.py
   ```
   atau python3 CreateFacebook2.py (Linux/macOS).
3. Saat pertama kali dijalankan, Device ID akan ditampilkan.
4. Jika belum terverifikasi, Anda akan otomatis dibawa ke WhatsApp admin.
5. Setelah disetujui, menu utama muncul:
   · [01] CREATE ACCOUNT → mulai membuat akun
   · [02] CHECK RESULT → lihat akun yang telah dibuat
   · [03] SETTINGS → coming soon

🔑 Lisensi & Verifikasi

Tools ini berbayar dan hanya bisa digunakan setelah Device ID Anda di-approve oleh admin.

Prosedur aktivasi:

1. Jalankan script → salin Device ID yang muncul.
2. Hubungi admin melalui WhatsApp di bawah.
3. Kirim Device ID dan selesaikan pembayaran.
4. Setelah di-approve, jalankan ulang script untuk langsung masuk ke menu utama.
